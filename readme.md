# Getting Started:  

While in client directory:
```bash
npm install
npm start
```

[Server/Admin Repo](https://github.com/la-flor/Gulfport_Server)

- Server/Admin repository is needed to retrieve events from database, but site will run without requiring a connection
- Server/Admin repository also has features to add, edit, and delete events for event page
